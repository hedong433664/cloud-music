# Cloud Music

## 项目介绍

本项目是一个音乐播放器，基于Vue3、TypeScript、Vite和Element Plus开发，实现了音乐播放、搜索、歌单等功能。

## 功能介绍

- 音乐播放：支持在线播放，支持暂停、播放、上一首、下一首等功能。
- 搜索：支持歌曲、歌手、专辑、用户等搜索。
- 用户：支持用户扫码登录等功能。

## 技术栈

- 前端：`Vue3` `Vite` `TypeScript`
- 后端：`NeteaseCloudMusicApi`

## 安装与运行

1. 克隆项目到本地

```bash
git clone https://github.com/hedong433664/cloud-music.git
```

2. 安装依赖

```bash
cd CloudMusic
pnpm install
```

3. 启动项目

```bash
pnpm run dev
```
